TSEG seguimiento de equipos
 
 Programa en desarrollo: Permite el seguimiento de cada equipo vendido por la firma LIE de manera interna, agrupando los equipos por cliente.
 Basado en HTML y Python. Permite registro de usuarios y blog según el tipo de usuario. Facilita el seguimiento de ventas, reparaciones y estado general de cada equipo.
 
 
 

Esta sección contiene información del programa para desarrolladores.

CMD:
pip install flask
pip install flask_wtf

if email error:::
pip install email_validator
